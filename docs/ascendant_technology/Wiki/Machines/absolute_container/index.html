Absolute Container

The Absolute Container is a late-game singular vault that combines massive item storage, energy buffering, and fluid storage in one block.

Machine Information​

SettingValue
Energy Capacity25,600,000 DE (25.6 MDE)
Fluid Capacity25,600,000 mB (25,600 buckets)
Storage Capacity168 slots (14×12)
Item OutputAutomatic transfer from the last 9 slots
Network IntegrationEnergy + fluid networks
Upgrade SlotsNone

How It Works​

  1. Place the Absolute Container and open its inventory.
  2. Insert items into the main 168-slot grid.
  3. Optionally connect cables/pipes for network integration.
  4. The last 9 slots can be used as an automated output buffer.

Notes​

  • The block can function as a central logistics hub for late-game bases.
  • It can receive and distribute both energy and fluids while storing items.
