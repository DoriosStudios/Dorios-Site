Liquifier

The Liquifier melts solid items into fluids and can produce optional residue byproducts.

Machine Information​

SettingValue
Energy Capacity8,192,000 DE (8.19 MDE)
Energy Consumption3,600–12,800 DE per craft
Base Processing Rate3,600 DE/t
Fluid Tank Capacity128,000 mB
Upgrade Slots2
Supported FluidsLiquified Aetherium, Dark Matter

How It Works​

  1. Insert a valid solid input item.
  2. Ensure free capacity in the fluid tank.
  3. Provide enough DE for the selected recipe.
  4. Extract fluid via pipes/capsules and collect byproducts when generated.
