# Item Components

UtilityCraft provides custom item components that allow you to define
special behavior directly inside the item JSON.

These components are used to extend machines and tools such as:

-   Autofisher
-   Autosieve
-   Hammers
-   Manual resource tools

Item components define scaling, tiers, multipliers, and behavior without
requiring additional scripting.

------------------------------------------------------------------------

# Available Item Components

-   [Fishing Net](fishing-net) --- Used by the Autofisher.
-   [Mesh](mesh) --- Used by the Autosieve (not compatible
    with Basic Sieve).
-   [Hammer](hammer) --- Allows manual crushing using Crusher
    recipes.
-   [Dig Pebble](dig-pebble) --- Enables pebble generation
    from dirt and grass.
