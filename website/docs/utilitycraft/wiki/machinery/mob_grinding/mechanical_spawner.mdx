### Block Information

| Setting | Value |
|----------|--------|
| **Energy Usage** | None |
| **Spawn Interval** | 500–700 ticks (random) |
| **Base Spawn Chance** | 60% per entity |
| **Supports Quantity Upgrades** | Yes (0–4) |
| **Destruction Time** | 3.0 seconds |

💡 **Tip:** Use **Quantity Upgrades** to spawn more mobs per cycle — at max level, you get 4 entities per tick with a 25% bonus chance.

### Supported Mob Types

The Mechanical Spawner supports **18 mob essences**:

- **Blaze Essence**
- **Chicken Essence**
- **Cow Essence**
- **Creeper Essence**
- **Enderman Essence**
- **Guardian Essence**
- **Hoglin Essence**
- **Illager Essence**
- **Magma Cube Essence**
- **Mooshroom Essence**
- **Pig Essence**
- **Piglin Essence**
- **Sheep Essence**
- **Skeleton Essence**
- **Slime Essence**
- **Spider Essence**
- **Wither Skeleton Essence**
- **Zombie Essence**

:::info Illager Essence
The **Illager Essence** spawns a weighted mix of illager types:  
- **Pillager** (45%), 
- **Vindicator** (35%), 
- **Witch** (15%), 
- **Evoker** (1%).
:::

### How to Use

1. Place the **Mechanical Spawner**.  
2. Hold a **Mob Essence** and **sneak + interact** to assign it.  
3. Interact with an empty hand to open the toggle menu and turn it **On**.  
4. The spawner will begin spawning the assigned mob at random intervals.

### Managing Essences

| Action | How |
|--------|-----|
| **Assign Essence** | Sneak + interact while holding an essence. |
| **Swap Essence** | Sneak + interact with a different essence (requires double-tap confirmation). |
| **Recover Essence** | Sneak + interact while holding a **Glass Bottle** to extract the current essence. |
| **Toggle On/Off** | Interact with an empty hand (not sneaking) to open the settings menu. |
