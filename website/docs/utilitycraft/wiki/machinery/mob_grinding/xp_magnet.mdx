### Block Information

| Setting | Value |
|----------|--------|
| **Energy Usage** | None |
| **Tick Interval** | Every 5 ticks |
| **Maximum Radius** | 11 blocks |
| **Placeable On** | All block faces (top, sides) |

💡 **Tip:** The **XP Magnet** only moves XP orbs. If you want XP fluid in a tank, use an **XP Drain** on top of a **Fluid Tank** and stand on the drain to convert your player XP.

### How to Use

1. Place the **XP Magnet** on any surface near where mobs die.  
2. Interact with an empty hand to open the settings menu.  
3. Toggle the magnet **On** and set your desired **Radius** (0–11 blocks).  
4. All XP orbs within range will be teleported to the magnet's position.

---

### Settings Menu

| Control | Description |
|---------|-------------|
| **Off/On** | Toggles the magnet active or inactive. |
| **Radius** | Slider from 0 to 11 blocks — controls how far orbs are pulled from. |
