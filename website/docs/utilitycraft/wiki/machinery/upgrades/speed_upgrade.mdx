## Where It Works

### Machine slot upgrades

Speed Upgrade works in upgrade slots of many powered machines (Crusher, Assembler, Auto Fisher, Electro Press, and more).

### Applied block upgrades

You can also apply it directly (using the upgrade item on the block) on:

- Item Exporters
- Fluid Extractors
- Mechanic Hopper / Mechanic Upper / Mechanical Dropper
- Ender Hopper
- Mechanical Spawner

---

## Practical Effect

- **Machines:** complete work faster.
- **Transport blocks:** shorter transfer intervals.
- **Spawner:** shorter spawn cycle intervals.

> Speed is very strong, but pairing it with Energy Upgrade is recommended to keep power usage under control.

---

## Machine Speed Levels (Slot System)

For machine slot upgrades, total Speed levels are capped at 8.

| Total Speed Level | Multiplier |
|---:|---:|
| 0 | x1.00 |
| 1 | x1.25 |
| 2 | x1.75 |
| 3 | x2.50 |
| 4 | x3.50 |
| 5 | x4.75 |
| 6 | x6.25 |
| 7 | x8.00 |
| 8 | x10.00 |
