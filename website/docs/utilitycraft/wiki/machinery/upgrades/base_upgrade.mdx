---

## What It Does

- Works as a **core crafting ingredient** for advanced upgrades.
- Has **no direct machine effect** when used alone.

---

## Used In These Upgrades

- Speed Upgrade
- Energy Upgrade
- Range Upgrade
- Filter Upgrade
- Smart Filter Upgrade
- Quantity Upgrade
- Damage Upgrade
