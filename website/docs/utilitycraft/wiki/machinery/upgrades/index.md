# Machine Upgrades

Upgrades are the easiest way to tune your UtilityCraft setup without rebuilding everything.

You can use upgrades in **two ways**:

- **Slot Upgrades**: inserted into machine upgrade slots (for example Speed and Energy).
- **Applied Upgrades**: hold the upgrade item and apply it directly to compatible blocks (for example Filter and Smart Filter).

---

## Upgrade List

| Upgrade | Main Purpose | Page |
|---|---|---|
| Base Upgrade | Crafting component for advanced upgrades | [Base Upgrade](./utilitycraft-base-upgrade) |
| Speed Upgrade | Makes machines and transport blocks run faster | [Speed Upgrade](./utilitycraft-speed-upgrade) |
| Energy Upgrade | Improves machine energy efficiency | [Energy Upgrade](./utilitycraft-energy-upgrade) |
| Range Upgrade | Increases working radius for specific blocks | [Range Upgrade](./utilitycraft-range-upgrade) |
| Filter Upgrade | Enables item/fluid filtering options | [Filter Upgrade](./utilitycraft-filter-upgrade) |
| Smart Filter Upgrade | Advanced item routing on Item Importers | [Smart Filter Upgrade](./utilitycraft-smart-filter-upgrade) |
| Quantity Upgrade | Increases Mechanical Spawner output | [Quantity Upgrade](./utilitycraft-quantity-upgrade) |
| Damage Upgrade | Increases Mob Grinder damage limit | [Damage Upgrade](./utilitycraft-damage-upgrade) |
| Ultimate Upgrade | Reserved upgrade (currently no gameplay effect) | [Ultimate Upgrade](./utilitycraft-ultimate-upgrade) |

---

## Quick Tip

If you're starting out:

1. Craft **Base Upgrades** first.
2. Use **Speed + Energy** on your core machines.
3. Add **Filter / Smart Filter** when your transport network gets larger.
