# Machine Upgrades

Upgrades are the easiest way to tune your UtilityCraft setup without rebuilding everything.

You can use upgrades in **two ways**:

- **Slot Upgrades**: inserted into machine upgrade slots (for example Speed and Energy).
- **Applied Upgrades**: hold the upgrade item and apply it directly to compatible blocks (for example Filter and Smart Filter).

---

## Upgrade List

| Upgrade | Main Purpose | Page |
|---|---|---|
| Base Upgrade | Crafting component for advanced upgrades | [Base Upgrade](./base_upgrade) |
| Speed Upgrade | Makes machines and transport blocks run faster | [Speed Upgrade](./speed_upgrade) |
| Energy Upgrade | Improves machine energy efficiency | [Energy Upgrade](./energy_upgrade) |
| Range Upgrade | Increases working radius for specific blocks | [Range Upgrade](./range_upgrade) |
| Filter Upgrade | Enables item/fluid filtering options | [Filter Upgrade](./filter_upgrade) |
| Smart Filter Upgrade | Advanced item routing on Item Importers | [Smart Filter Upgrade](./smart_filter_upgrade) |
| Quantity Upgrade | Increases Mechanical Spawner output | [Quantity Upgrade](./quantity_upgrade) |
| Damage Upgrade | Increases Mob Grinder damage limit | [Damage Upgrade](./damage_upgrade) |
| Ultimate Upgrade | Reserved upgrade (currently no gameplay effect) | [Ultimate Upgrade](./ultimate_upgrade) |

---

## Quick Tip

If you're starting out:

1. Craft **Base Upgrades** first.
2. Use **Speed + Energy** on your core machines.
3. Add **Filter / Smart Filter** when your transport network gets larger.
