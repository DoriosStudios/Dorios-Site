---

## Where It Works

- **Item Importer** (all color variants)

---

## What It Unlocks

- Slot-aware routing rules for incoming items
- Better control for machines that need inputs in exact slots
- Cleaner automation loops with fewer jams

> Same interface style, but with advanced routing options enabled.

---

## Practical Use Case

If a machine needs:

- Item A in slot 0
- Item B in slot 1
- Item C in slot 2

Smart Filter lets your importer keep this organized automatically.
