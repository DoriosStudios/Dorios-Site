## Where It Works

Apply directly on compatible blocks:

- Item Exporter
- Item Importer
- Fluid Extractor
- Mechanic Hopper
- Mechanic Upper
- Mechanical Dropper
- Ender Hopper

---

## What It Unlocks

Depending on the block, the same UI gains extra options like:

- **Whitelist mode** (allow only listed entries)
- **Blacklist mode** (block listed entries)
- Add/remove items or fluids from filter lists

> The interface layout stays the same. Filter Upgrade unlocks additional actions in that existing menu.

---

## How to Apply

1. Hold a **Filter Upgrade** in your main hand.
2. Interact with a compatible block.
3. The block gains filter functionality.
