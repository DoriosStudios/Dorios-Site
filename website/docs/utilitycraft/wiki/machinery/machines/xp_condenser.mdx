### Overview

This machine acts as a **XP storage unit**, converting your XP into an internal fluid form that can be stored in **Fluid Tanks**.  
You can safely store large amounts of XP without risk of losing it on death.

---

### Machine Information

| Setting | Value |
|----------|--------|
| **Energy Usage** | None |
| **Storage Method** | Fluid Tanks |
| **Operation Type** | Manual (player interaction) |

💡 **Tip:** Pair the XP Condenser with a **Tank System** to create mobile XP storage or automation setups that use XP as a fluid.

## How It Works

Players can **store** or **extract** XP by clicking buttons on the interface.  
Internally, XP is managed using **fluid tanks**.

1. Insert a **Fluid Tank** by interacting with the XP Condenser.  
2. Deposit or withdraw XP using the interface buttons:  
   - **+1 / +10 / +Max** to store XP  
   - **−1 / −10 / −Max** to withdraw XP  
3. Retrieve the tank anytime by **sneaking and interacting** with the machine.

---

### Example Uses

| Use Case | Description |
|-----------|--------------|
| XP Storage | Keep XP safe between play sessions |
| Machine Fuel | Power machines that use XP as a resource |
| Transportable XP | Store XP fluid in tanks and move them elsewhere |

---

### Interface Buttons

| Button | Function |
|---------|-----------|
| **+1 / −1** | Add or remove 1 level |
| **+10 / −10** | Add or remove 10 levels |
| **+Max / −Max** | Deposit or withdraw all XP instantly |

---

### How to Use

1. Place the **XP Condenser** in your base.  
2. Insert a **Fluid Tank** to enable XP storage.  
3. Use the interface to **deposit or withdraw** XP levels.  
4. Sneak-click to remove the tank when needed.
