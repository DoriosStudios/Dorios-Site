The **Seed Synthesizer** automates crop and sapling growth loot by consuming Dorios Energy.

It uses the plant registry from `BP/scripts/config/recipes/plants.js`, so drops and base costs follow the same data used by the addon runtime.

---

## Machine Information

| Setting | Value |
|---|---|
| Energy Cap | 2,048,000 DE |
| Base Rate | 160 |
| Upgrade Slots | 2 (slots 4 and 5) |
| Seed Input | Slot 3 |
| Soil Input | Slot 6 |
| Output Range | Slots 7–15 |

> Source: `BP/blocks/machinery/machines/seed_synthetizer.json`

---

## Energy Cost Rules

Final cycle cost is:

`Final Cost = Plant Base Cost × Soil Cost Multiplier`

### Common Base Cost Tiers

| Tier | Base Cost |
|---|---|
| Vanilla crops / trees / common plants | 8,000 DE |
| BC Tier 1 crops | 64,000 DE |
| BC Tier 2 crops | 128,000 DE |
| BC Tier 3 crops | 512,000 DE |
| BC Tier 4 crops | 1,024,000 DE |

## How to Use

1. Insert a valid seed or sapling in slot 3.
2. Insert a valid soil block in slot 6.
3. Keep enough Dorios Energy in the machine.
4. (Optional) Add upgrades to improve throughput.
5. Collect outputs from slots 7–15.

If the machine is not running, check the in-game warnings (`No Seed`, `No Soil`, `Invalid Soil`, `Invalid Seed`, `Output Full`, `No Energy`).
