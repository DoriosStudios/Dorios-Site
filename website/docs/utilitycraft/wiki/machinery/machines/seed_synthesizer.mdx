It is designed for passive farming setups and becomes much more efficient when paired with higher-tier soils.

### Machine Information

| Setting | Value |
|----------|--------|
| **Energy Cap** | 2,048,000 DE |
| **Base Processing Rate** | 160 DE/t |
| **Accepts Speed Upgrades** | Yes |
| **Overclockable** | No |

💡 **Tip:** Higher soil tiers reduce final cost and increase processing speed.

## Energy Cost Behavior

Each cycle cost depends on:

- The base cost of the selected seed/plant category
- The cost multiplier of the inserted soil

In practice, better soils make the machine faster and cheaper per completed cycle.

## Common Warnings

If the machine stops, check these messages:

- No Seed
- No Soil
- Invalid Soil
- Invalid Seed
- Output Full
- No Energy

---

### How to Use

1. Place the **Seed Synthesizer** in your world.
2. Insert a valid seed or sapling into Slot 3.
3. Insert a valid soil block into Slot 6.
4. Ensure enough stored energy.
5. (Optional) Add upgrades for higher throughput.
6. Collect outputs from Slots 7 to 15.
