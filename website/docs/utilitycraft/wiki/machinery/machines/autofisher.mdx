### Machine Information

| Setting | Value |
|----------|--------|
| **Energy Cap** | 512,000 DE |
| **Energy Cost per Operation** | 6,400 DE |
| **Base Processing Rate** | 40 DE/t |
| **Accepts Speed Upgrades** | Yes |
| **Overclockable** | No |

💡 **Tip:** Better net tiers improve loot quality and can increase the number of items per cycle.

## Requirements to Operate

To run correctly, the Auto Fisher needs:

- A valid **Fishing Net** in the net slot
- Nearby **water** (or a **Sink**) around the machine
- Enough stored **Dorios Energy**
- Free space in output slots

---

## How It Works

1. Insert a **Fishing Net** in Slot 6.
2. Place water near the machine (or use a Sink setup).
3. Keep the machine powered.
4. The Auto Fisher performs cycles and sends generated loot to output slots.

---

## Net Effects

The equipped Fishing Net controls important machine behavior:

- Processing speed
- Loot chance multiplier
- Item amount multiplier
- Rolls per cycle
- Tier and luck for higher-quality drops

---

## Common Warnings

If the machine stops, check these messages:

- No Net Item
- Need Water!
- No Energy
- Output Full

---

### How to Use

1. Place the **Auto Fisher** in your world.
2. Add a valid **Fishing Net**.
3. Ensure nearby water (or Sink support).
4. Supply enough energy (**6,400 DE per operation**).
5. Collect loot from output slots.
