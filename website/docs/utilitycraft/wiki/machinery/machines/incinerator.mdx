### Machine Information

| Setting | Value |
|----------|--------|
| **Energy Cap** | 16,000 DE |
| **Energy Cost per Operation** | 800 DE |
| **Base Processing Rate** | 20 DE/t |
| **Accepts Speed Upgrades** | Yes |
| **Overclockable** | No |

💡 **Tip:** The Incinerator can accept **Speed Upgrades** to increase smelting speed — up to 4 items per second at maximum speed.

| Speed Level | Ticks per Smelt | Time per Item (Seconds) |
|--------------|-----------------|--------------------------|
| 0 (No Upgrades) | 80 | 4 |
| 1 | 40 | 2 |
| 2 | 20 | 1 |

### How to Use

1. Place the Incinerator in your world.  
2. Insert an accepted item into the input slot.  
3. Ensure it has enough stored energy (**800 DE per operation**).  
4. (Optional) Add **Speed Upgrades** to increase smelting speed.
