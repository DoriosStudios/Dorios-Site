## Channel system

Wireless linkage uses a shared <b>3-color channel</b>.

Example channel: <b>white / white / white</b>

A transmitter only sends power to receivers on the exact same channel.

---

## Wrench Configuration Flow

Using the <b>Wrench</b>:

- **Normal click on a receiver** opens node settings with:
	- Node Mode ("Receiver" / "Transmitter")
	- Transfer mode ("Nearest" / "Farthest" / "Round")
- **Normal click on a transmitter** quickly toggles block mode.
- **Sneak click on receiver/transmitter** opens channel settings to set the 3-color channel.

---

## Runtime behavior

### Transmitter

- finds matching receivers in range on the same channel
- sends up to its per-tick rate budget
- shows transferring amount in label UI

### Receiver

- receives from transmitters (same channel)
- outputs energy to adjacent energy network using transfer logic
