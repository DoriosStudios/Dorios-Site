## Variants

| Block | Primary role |
|---|---|
| Mechanic Hopper | Pull from above, output to side/down |
| Mechanic Upper | Pull from below, output up/side |
| Mechanical Dropper | Drops items into world directionally |
| Ender Hopper | Vacuum pickup in configurable radius |

---

## Tick Intervals by Speed State

### Mechanic Hopper / Upper / Dropper

- 0 = 5t
- 1 = 4t
- 2 = 3t
- 3 = 2t
- 4 = 1t

### Ender Hopper

- 0 = 10t
- 1 = 8t
- 2 = 6t
- 3 = 4t
- 4 = 2t

---

## Script-backed Behavior Details

- Shared filter model:
	- Filter mode can be enabled with upgrades
	- Whitelist and blacklist are both supported
- Hopper and Upper can optionally pull from nearby minecarts (`minecraft:chest_minecart` / `minecraft:hopper_minecart`):
	- minecart pulling can be toggled in settings
- Ender Hopper:
	- collects dropped items in configurable radius
	- pickup radius can be configured
	- can be toggled ON/OFF (`isOff`)

---

## Interaction / UI

- With filter upgrade, hopper-style blocks open a filter menu for:
	- whitelist/blacklist mode
	- minecart pull toggle
	- add/remove filter entries
- Sneak-interacting an Ender Hopper opens a settings modal:
	- enabled toggle
	- pickup radius slider
