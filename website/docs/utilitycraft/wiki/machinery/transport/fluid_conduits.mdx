| Block | In-game name | UI |
|---|---|---|
| Fluid Pipe | Fluid Pipe (Default) | No |
| Fluid Extractor | Fluid Extractor (Default) | Yes |

> As requested, the pipe itself has no UI. The configurable block is the extractor.

---

## Supported Source Types (Extractor)

From runtime logic in `fluid_extractor`:

- UtilityCraft fluid containers
- Full vanilla source blocks:
  - Water
  - Lava
- Crucible (lava levels)
- Sink (infinite water)

---

## Modes, Speed, and Throughput

- Transfer mode: nearest, farthest, and round-robin
- Round-robin mode rotates target priority over time
- Tick interval by speed state:
  - 0 = 10t
  - 1 = 8t
  - 2 = 6t
  - 3 = 4t
  - 4 = 2t
- Runtime transfer budget starts at `4000 mB` per execution cycle.

### Crucible Rule

- Crucible extraction works in `250 mB` steps.
- Draining updates the internal lava level based on the amount actually moved.

---

## Extractor UI Features

The extractor menu includes:

- ON/OFF toggle
- transfer mode selector
- fluid filter list view
- add fluid type from mainhand fluid container
- add fluid type from source block in front
- remove configured fluid entries
