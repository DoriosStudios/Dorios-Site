## Script-backed Behavior

- Energy cables participate in transfer-network scanning logic in <code>BP/scripts/machinery/transfer_system/system.js</code>.
- They use the shared energy/tube network model and support color-based subnetworks.
- Their shape updates by direction to visually show connected neighbors.
- The cable itself has <b>no UI</b>; transfer limits are defined by producers/consumers and transport logic, not by a cable menu.

---

## Technical Data

| Property | Value |
|---|---|
| In-game name | Energy Cable |
| Collision | Invisible collision box (`0,0,0`) |
| Core role | Physical energy path + network connectivity |
| Subnetworks | Color-based |
| UI | None |
