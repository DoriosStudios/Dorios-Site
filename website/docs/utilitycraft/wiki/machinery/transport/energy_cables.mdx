# Energy Cables

`Energy Cable` (`utilitycraft:energy_cable`) is the backbone of wired DE distribution.

## Role in network

- physically connects generators, machines, batteries, and compatible energy nodes
- participates in energy network scans/rescans
- supports color-tag network separation (`dorios:color.*`)

## Technical notes

- cable itself has no user UI
- throughput is governed by source/receiver blocks and transfer logic, not by a fixed cable-tier limit
- cable geometry updates by neighbor connection states for visual network feedback

## When to use

- medium and long-distance power lines
- clean machine rows with centralized generation
- decoupling producers from consumers while keeping deterministic wiring
