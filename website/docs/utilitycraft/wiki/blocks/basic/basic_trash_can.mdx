# Basic Trash Can

The <b>Basic Trash Can</b> permanently destroys any item inserted into it.
Use it as a safe overflow endpoint for automation lines that generate junk.

## What It Does

- Deletes inserted items automatically.
- Prevents storage systems from clogging with unwanted drops.
- Works without energy.

---

## Important Warning

Items sent here are gone forever.

---

## Common Uses

- Remove junk drops from mob farms.
- Discard low-value byproducts from machines.
- Keep your main storage clean and focused.
