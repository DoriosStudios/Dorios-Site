# Basic Trash Can

The **Basic Trash Can** permanently destroys any item inserted into it.

Think of it as a safe overflow endpoint for automation lines.

---

## What It Does

- Deletes inserted items automatically.
- Prevents storage systems from clogging with unwanted drops.
- Works without energy.

---

## Basic Info

| Setting | Value |
|---|---|
| Inventory Size | 27 slots |
| Energy Use | None |
| Tick Interval | Every 10 ticks |

---

## Important Warning

Items sent here are gone forever.

Only connect this block to filtered outputs or true trash routes.

---

## Common Uses

- Remove junk drops from mob farms.
- Discard low-value byproducts from machines.
- Keep your main storage clean and focused.
