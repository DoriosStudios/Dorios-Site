# Basic Utility Blocks

These are practical early-game blocks that help your base run smoother.

They don't need Dorios Energy to work, and most are simple to use right away.

---

## Included Blocks

- [Sieve](./utilitycraft-sieve)
- [Crucible](./utilitycraft-crucible)
- [Sink](./utilitycraft-sink)
- [Basic Trash Can](./basic-trash-can)
- [Big Torch](./utilitycraft-big-torch)
- [Lantern](./utilitycraft-lantern)

---

## Quick Start Order (Suggested)

If you're just getting started:

1. Place a **Sieve** for manual resource progression.
2. Add a **Crucible** to generate lava from basic blocks.
3. Use a **Sink** for endless water loops.
4. Add **Big Torch / Lantern** for safer work areas.
5. Finish with a **Basic Trash Can** to remove junk items.
