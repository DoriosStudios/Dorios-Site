# Basic Utility Blocks

These are practical early-game blocks that help your base run smoother.

They don't need Dorios Energy to work, and most are simple to use right away.

---

## Included Blocks

- [Sieve](./utilitycraft-sieve) — manual progression and resource filtering.
- [Crucible](./utilitycraft-crucible) — converts blocks into lava.
- [Sink](./utilitycraft-sink) — infinite water source for automation.
- [Basic Trash Can](./basic-trash-can) — permanently deletes unwanted items.
- [Big Torch](./utilitycraft-big-torch) — medium large-area light coverage.
- [Lantern](./utilitycraft-lantern) — extra-large area light coverage.

---

## Suggested Setup Order

If you're just getting started:

1. Place a **Sieve** for manual resource progression.
2. Add a **Crucible** to generate lava from basic blocks.
3. Use a **Sink** for endless water loops.
4. Add **Big Torch** and **Lantern** for safer large work areas.
5. Finish with a **Basic Trash Can** to remove junk items.

---

## Lighting Coverage Quick Note

- **Big Torch:** projects light points up to **14 blocks** away.
- **Lantern:** projects the same pattern from 4 anchors at ±19 blocks, reaching up to **33 blocks** from the lantern.
