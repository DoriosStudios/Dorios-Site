# Lantern

The UtilityCraft <b>Lantern</b> is a wide-area lighting block.
It projects multiple light clusters for large halls, corridors, and automation floors.

## Why It Helps

- Covers larger spaces more effectively.
- Great for open floors and long pathways.
- Keeps utility areas safer and easier to navigate.

---

## Coverage Area

Lantern coverage is projected as 4 offset light clusters (not a full filled circle).

| Coverage Metric | Value |
|---|---|
| **Anchor offsets** | `(19, 0)`, `(-19, 0)`, `(0, 19)`, `(0, -19)` |
| **Per-anchor pattern** | Same 40-point pattern as Big Torch |
| **Farthest point from lantern center** | `19 + 14 = 33` blocks |
| **Projected light level** | 15 |

This helps maintain stronger practical illumination in bigger builds.

---

## Recommended Use Cases

- Long corridors
- Open automation floors
- Perimeter lighting where one point-light is not enough
