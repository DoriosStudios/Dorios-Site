# Lantern

The UtilityCraft **Lantern** is a large-area lighting utility block.

## Extended light projection

In addition to its block light, the script system applies projection anchors around the lantern to keep nearby areas lit.

Current anchor pattern is configured around cardinal offsets in the light block runtime.

> Source: `LIGHT_CONFIG` for `utilitycraft:lantern` in `BP/scripts/blocks/light_blocks.js`

## Practical use

- Long corridors
- Open automation floors
- Perimeter lighting where one point-light is not enough
