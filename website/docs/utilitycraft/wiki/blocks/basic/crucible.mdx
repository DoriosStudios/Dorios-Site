## How It Works

1. Place a heat source under the Crucible.
2. Insert a valid solid block.
3. Wait while it melts into lava.

Lava is generated in **250 mB steps**.

---

## Valid Inputs

The Crucible accepts common stone-like materials:

- Cobblestone / Stone / Diorite / Granite / Blackstone → `1` unit
- Netherrack → `4` units

---

## Quick Notes

- Maximum stored lava: `1000 mB`.
- Works with **Fluid Extractors** for automation.
- Lava transfer happens in **250 mB** steps.
