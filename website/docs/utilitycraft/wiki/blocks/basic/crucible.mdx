# Crucible

The **Crucible** turns stone-type blocks into lava.

It is one of the most useful basic blocks for starting fluid-based automation.

---

## How It Works

1. Place a heat source under the Crucible.
2. Insert a valid solid block.
3. Wait while it melts into lava.

Lava is generated in **250 mB steps**.

---

## Valid Inputs

The Crucible accepts common stone-like materials, including:

- Cobblestone / Stone / Diorite / Granite / Blackstone: `1`
- Netherrack: `4`

Netherrack provides stronger value per item, so it's efficient for lava production.

---

## Capacity and safety rules

- Lava state max: `4` (1,000 mB total)
- Internal input handling respects remaining capacity.

This helps avoid accidental overflow behavior.

---

## Automation Compatibility

You can pull Crucible lava using **Fluid Extractors**.

Important runtime behavior:

- transfer is quantized to **250 mB steps**
- drain is proportional to actual transferred amount

Perfect for feeding generators or lava-consuming machine loops.
