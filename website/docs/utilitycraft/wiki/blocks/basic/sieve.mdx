## How to Use

1. Insert a valid mesh item.
2. Insert a valid input block.
3. Interact repeatedly to process it.
4. At the final stage, drops are generated.

---

## Mesh Tiers

The block supports these mesh states:

- String Mesh
- Flint Mesh
- Copper Mesh
- Iron Mesh
- Golden Mesh
- Emerald Mesh
- Diamond Mesh
- Netherite Mesh

Higher-tier meshes generally improve your results.

---

## Inputs and Drops

Accepted inputs and drop pools are defined by UtilityCraft recipes.

This lets the Sieve support many progression paths, especially in early game.

---

## Practical Tips

- Sneak + empty hand can remove the mesh when the block is idle.
- Keep multiple sieves nearby to process faster as you expand.
- Use better meshes when targeting rarer materials.
