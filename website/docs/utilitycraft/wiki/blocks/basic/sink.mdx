# Sink

The **Sink** is an infinite water utility block used for automation and fluid extraction setups.

## Player interaction

Default interaction behavior:

- Empty bucket → gives water bucket
- Water bucket → gives empty bucket

> Source: `utilitycraft:sink` handler in `BP/scripts/blocks/on_interact.js`

## Automation interaction

The fluid extractor system recognizes Sink as an **infinite water source**.

- extractor can pull water continuously from Sink
- source is treated as non-depleting in runtime

> Source: `fluid_extractor` source handling in `BP/scripts/machinery/transfer_system/system.js`

## Typical use cases

- Feeding fluid tank lines
- Supplying machines requiring water-based loops
- Stable starter fluid infrastructure
