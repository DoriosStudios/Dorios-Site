# Sink

The **Sink** is an infinite water source block.

It's one of the easiest ways to keep water-based automation running forever.

---

## Player Interaction

Basic interactions:

- Empty bucket → gives water bucket
- Water bucket → gives empty bucket

---

## Automation Use

Fluid systems treat Sink as an **infinite source**.

That means you can pull water continuously with Fluid Extractors.

---

## Common Uses

- Feeding fluid tank lines
- Supplying machines requiring water-based loops
- Stable starter fluid infrastructure
