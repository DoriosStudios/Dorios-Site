The **Enchantment Station** is an enchantment-focused machine currently represented in data as `utilitycraft:enchantment_station`.

> [!NOTE]
> Render, UI, and recipe images are not available yet in the docs assets.

---

## Machine Information

| Setting | Value |
|----------|--------|
| **Block Identifier** | `utilitycraft:enchantment_station` |
| **Energy Capacity** | 25,600,000 DE |
| **Energy Cost (base)** | 8,000 DE |
| **Base Processing Rate** | 64,000 DE/t |
| **Fluid Tank Capacity** | 128,000 mB |
| **Upgrade Slots** | 3 (slots 19, 20, 21) |
| **Inventory Size** | 32 |
