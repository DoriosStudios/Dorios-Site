<h1 align="center">Ascendant Technology – Machines</h1>

<p align="center">
  Explore the machine lineup of Ascendant Technology.  
  Many of these are unique, high-end machines designed for advanced automation, but there are also upgraded versions of standard machines.
</p>

---

## Machines Overview

### [Absolute Container](./absolute_container)
Massive late-game storage block for items, energy, and fluids.

### [Enchantment Station](./enchantment_station)
Enchantment-focused processing station (early documentation pass).

### [Catalyst Weaver](./catalyst_weaver)
Multi-catalyst precision crafting with optional fluid requirements.

### [Cryo Chamber](./cryo_chamber)
Three-in-one machine for stabilization, cooling, and Cryofluid generation.

### [Duplicator](./duplicator)
Template-based item replication using Liquified Aetherium.

### [Energizer](./energizer)
Converts base materials into energized variants using DE only.

### [Laser Barrier](./laser_barrier)
Defensive field projector for area denial and base protection.

### [Liquifier](./liquifier)
Converts solids into advanced fluids with optional byproducts.

### [Network Center](./network_center)
Energy network scanner and real-time diagnostics hub.

### [Overclock Network](./overclock_network)
Relay + Tower overclock infrastructure for factory-wide machine boosting.

### [Residue Processor](./residue_processor)
Reclaims useful materials from residue-type inputs.

### [Singularity Fabricator](./singularity_fabricator)
Extreme end-game duplicator powered by Dark Matter.

### [Vaporworks Processor](./vaporworks_processor)
Fluid-to-fluid processing machine for advanced conversion chains.
