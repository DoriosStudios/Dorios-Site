The **Catalyst Weaver** performs advanced multi-input processing with a base item, up to six catalysts, and optional fluid support.

---

## Machine Information

| Setting | Value |
|----------|--------|
| **Energy Capacity** | 256,800,000 DE (256.8 MDE) |
| **Energy Consumption** | 5,400–601,600 DE per craft |
| **Base Processing Rate** | 180 DE/t |
| **Fluid Tank Capacity** | 128,000 mB |
| **Upgrade Slots** | 3 |

## How It Works

1. Insert one base item in the primary input.
2. Fill catalyst slots with the exact required combination.
3. Add required fluid when applicable.
4. Start processing and collect main output and optional byproduct.

---

## Supported Fluids

- **Lava**
- **Liquified Aetherium**
- **Dark Matter**
