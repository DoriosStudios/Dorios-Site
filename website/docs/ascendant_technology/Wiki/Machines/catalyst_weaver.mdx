## How It Works

1. Insert one base item in the primary input.
2. Fill catalyst slots with the exact required combination.
3. Add required fluid when applicable.
4. Start processing and collect main output and optional byproduct.

### Recipe reading

- The **base item**, **catalyst set**, and optional **fluid** must all match the recipe.
- Keep the byproduct slot clear during longer sessions.

## Supported Fluids

- **Cryofluid**

- **Lava**
- **Liquified Aetherium**
