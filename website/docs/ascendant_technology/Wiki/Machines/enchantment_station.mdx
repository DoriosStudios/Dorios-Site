| Disenchant Books | Book handling slot |
| Output | Final result collection |

## How It Works

1. Insert items into the relevant enchant/disenchant input groups.
2. Add catalyst when required.
3. Ensure fluid and DE are available.
4. Collect processed items from output slots.
